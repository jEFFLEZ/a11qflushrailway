# @funeste38/spyder

Local assistant primitives for the Funesterie stack.

This package is the small Spyder layer used by QFLUSH-oriented tooling when you need a lightweight server, a simple in-memory runtime and an explicit bridge to A11 or another orchestrated service.

## Install

```bash
npm install @funeste38/spyder
```

## Development

```bash
npm install
npm run build
```
